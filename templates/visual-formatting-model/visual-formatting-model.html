Visual Formatting Model

The box model

The visual formatting model transforms each element of a Web document into one or more boxes that conform to the CSS box model.

Every box has a core structure that extends to a measurable edge or boundary. The dimensions of a box can be precisely defined or constrained. A box can be given a positioning scheme in combination with a specific layout mode. These properties affect how a box interacts with surrounding boxes and how child elements interact.

Box-sizing is a non-inherited property.

*, *:before, *:after { box-sizing: border-box; } * { implies: inherit; }

border-box

padding and border will be inside of the box

box-sizing: border-box

content-box

padding, border & margin will be outside of the box

box-sizing: content-box
